# Divergence Meter（ニキシー管時計）

Steins;Gate のダイバージェンスメーターを模した、ニキシー管 **IN-14 × 8桁** の時計プロジェクト。
マイコンに **Seeed Studio XIAO ESP32-C3** を使い、5V 電源から高電圧を昇圧してニキシー管を駆動する。

## 特徴 / 仕様

- **表示管**: IN-14 × 8桁（ダイバージェンスメーター風の見た目）
- **点灯方式**: ダイナミック点灯（1/4 多重化：2グループ×4スロット、74141×2）
- **マイコン**: Seeed Studio XIAO ESP32-C3（5V / USB Type-C 給電、WiFi で NTP 同期）
- **高電圧電源**: MC34063A ベースの昇圧コンバータで 5V → 約 180V（外付け FET）
- **給電**: モバイルバッテリー（USB 5V）で持ち運び可
