# hardware（KiCad プロジェクト）

ダイバージェンスメーターの回路図・基板設計（KiCad 10）。
プロジェクト全体の概要は [../README.md](../README.md) を参照。

## 構成

```
hardware/
  divergence-meter.kicad_pro / .kicad_sch / .kicad_pcb   # 本体（コミット対象）
  sym-lib-table / fp-lib-table                           # プロジェクトのライブラリ登録
  lib/
    custom.kicad_sym       # 自作シンボル（+180V 電源シンボル 等）→ コミット
    custom.pretty/         # 自作フットプリント（.kicad_mod）→ コミット（現状は空）
    third-party/           # 外部入手の部品（ライセンス確認の上で同梱）
      NOTICE.md            # 同梱物の出典・ライセンス記録
```

> `*.kicad_prl`（個人ごとの表示設定）と `.history/`、各種バックアップ／自動保存ファイルは
> `.gitignore` で除外している。

回路図は階層シート構成（トップ → `power` / `control` / `drivers` / `tubes`）。
シート間は電源シンボル（`+5V` / `+3.3V` / `+180V` / `GND`）とグローバルラベルで接続している。

## ライブラリの登録について

`custom` ライブラリは `sym-lib-table` / `fp-lib-table` に **登録済み** なので、
プロジェクトを開けばそのまま使える（Symbol/Footprint Editor で `custom` を選択）。

外部部品を追加するときは:

1. 入手元のライセンスを確認（[../LIBRARIES.md](../LIBRARIES.md) 参照）。
2. 再配布可なら `lib/third-party/` に置き、`lib/third-party/NOTICE.md` に記録。
3. `sym-lib-table` / `fp-lib-table` に行を追加して登録。

## 高圧電源（5V → 180V 昇圧）

ダイナミック点灯では各スロットで **2 桁が同時点灯**（グループ A・B から 1 桁ずつ、74141×2）
するため、HV 側は常時 2 桁ぶんの電流が流れる。約 5mA（2.5mA×2、マーキング電流込みで最大 7mA、
桁切替の重なりを見込んで最大 10mA）。

- HV 出力電力: 180V × 10mA ≈ 1.8W
- 効率 80% で入力 ≈ 2.25W → **5V 換算で約 0.45A**（USB 給電で十分）

高圧は **MC34063A + 結合インダクタ（フライバック）** で生成する。USB の 5V のまま 180V を作るため、
巻数比 **1:10** の結合インダクタ T1 を使い、昇圧比をデューティではなく巻数比で稼ぐ。これにより:

- スイッチ OFF 時の一次側電圧 ≈ `5 + 180/10 = 約 23V` に収まり、**MC34063 の内蔵スイッチ
  (40V / 1.5A) をそのまま一次スイッチに使える**（外付け FET 不要 → 5V でのゲート駆動問題が消滅）。
- 昇圧比をデューティに頼らないので動作が安定する。

漏れインダクタによるターンオフ・スパイクから内蔵スイッチを守るため、一次側に
**ツェナークランプ（高速ダイオード＋ツェナー）**を入れる。

### 回路図

![5V→180V フライバック昇圧回路（MC34063A・1:10）](../docs/flyback-converter.svg)

> 青 = +5V 系 / 赤 = +180V 系 / 破線 = FB / T1 = 結合インダクタ 1:10

### 昇圧回路（MC34063A フライバック構成）

| 部品 | 値 / 型番 | 備考 |
|---|---|---|
| コントローラ | MC34063A | DIP-8（ソケット可）。内蔵スイッチ(40V/1.5A)を一次スイッチに使用。基準 1.25V |
| 結合インダクタ T1 | 1:10 / 一次 25µH / Isat 1.3A | Coilcraft **LPR6235-253**（SMD、巻線間絶縁 300Vrms）。巻数比で 180V を稼ぐ |
| タイミング Ct | 約 390〜470pF | CT(3)。発振周波数を決める（MC34063 は最大 ~100kHz）。要実測調整 |
| 整流ダイオード D | UF4007 / ES1J | 二次側。**高速整流必須**（一般整流は不可） |
| ツェナークランプ | Dsn(UF4007/高速) + DZ(27〜30V, ≥1W) | 一次側。スイッチノード→Vin にクランプ。ピーク ≈ 5+Vz < 40V。Vz 固定で実測不要 |
| 出力コンデンサ C_out | 4.7〜10µF / **250V** + 100nF/250V | 耐圧マージン必須 |
| 入力コンデンサ C_in | 低ESR 10〜22µF + 100nF | 一次ピーク電流を供給 |
| 電流検出 Rsc | **0.3Ω** / 0.5W | **VCC(6)〜Ipk(7) 間**。0.3V で一次ピークを約 1A に制限 |
| FB 分圧 | R1 ≈ **1.8MΩ** : R2 ≈ **12.5kΩ** | FB(5) へ。出力 ≒ 181V。R2 は半固定込みで校正 |
| ブリーダ抵抗 Rb | 470kΩ / 2W | 電源 OFF 後の放電用（**感電防止・必須**） |

**MC34063A 全8ピンの接続（DIP-8、開放ピンなし）:**

| Pin | 名称 | 接続先 |
|---|---|---|
| 1 | SWC（Switch Collector） | T1 一次巻線（スイッチノード） |
| 2 | SWE（Switch Emitter） | GND（スイッチ電流の戻り） |
| 3 | CT（Timing Capacitor） | Ct（約390pF）→ GND |
| 4 | GND | GND |
| 5 | FB（Comparator Inv. Input） | R1/R2 分圧タップ（基準 1.25V） |
| 6 | VCC | +5V（Rsc の入力側） |
| 7 | Ipk（Ipk Sense） | Rsc の負荷側（T1 一次側） |
| 8 | DrvC（Driver Collector） | VCC（+5V） |

> 内蔵スイッチ（SWC(1)–SWE(2)）が T1 の一次巻線を**直接スイッチ**する（外付け FET 無し）。
> OFF 時の一次電圧は約 23V（=5+180/10）で内蔵スイッチ(40V)の定格内、漏れスパイクは一次側
> ツェナークランプ（Dsn+DZ）でクランプ。過電流検出 Rsc は **VCC(6)–Ipk(7) 間**に入れ、`V(6)−V(7)` が約 0.3V で
> 一次ピーク制限。発振はタイミングコンデンサ Ct を **CT(3)** に接続して設定。
> 8ピンすべてに機能があり、**開放してよいピンは無い**。

### 主要定数の根拠

**出力電圧（R1 / R2）** — `Vout = 1.25V × (1 + R1/R2)`

```
R1/R2 = Vout/1.25 - 1 = 180/1.25 - 1 = 143
R1 = 1.8MΩ, R2 = 12.5kΩ → Vout = 1.25 × (1 + 144) ≈ 181V
```

- R2 は半固定抵抗込み（例: 10kΩ + 5kΩトリマ）で 170〜190V を実測校正。
- R1 には 180V がかかるため、910kΩ×2本直列など**耐電圧を確保した分割**にする。
- 下側 R2 は基準 1.25V に合わせて約 12.5kΩ。半固定の中心値もこれに合わせる。

**巻数比とスイッチ電圧** — `一次OFF電圧 ≈ Vin + Vout/N`

```
N=10（1:10）→ 5 + 180/10 = 約 23V（定常）
反射 V_R = 180/10 = 18V < Vz < 35V → Vz=27V を選ぶと
スパイクのピーク = 5 + 27 = 32V < 40V（内蔵スイッチ定格内）
```

**一次ピーク電流制限（Rsc）** — `Ipeak = 0.3V / Rsc`

```
一次ピークを約 1A に設定 → Rsc = 0.3V / 1A = 0.3Ω（< Isat 1.3A）
```

**発振タイミング（Ct）** — MC34063 は Ct で on 時間を決める

```
Ct ≈ 4.0×10⁻⁵ × t_on[s]   （データシート式）
t_on ≈ 9µs を狙うと Ct ≈ 390pF（数十kHz 動作、最大 ~100kHz 以内）
```

> 電力収支: HV 負荷は多重化で**常時 2 管ぶん**（通常 約 5mA / 0.9W、最大でも ~1.8W）。
> LPR6235-253（一次 25µH / Isat 1.3A）の DCM 伝送電力 `½·Lp·Ipk²·f` で十分まかなえる
> （通常は余裕、フル輝度時のみ上限寄り）。**8 管同時の静的点灯は対象外**（多重化前提）。

### 各桁のアノード抵抗

出力 180V、管維持電圧 ≈ 150V 前提で 2.5mA に制限:

```
R = (180 - 150) / 2.5mA ≈ 12kΩ（10〜15kΩ, 1/2W）
```

ダイナミック点灯のため、各桁のアノードごとに 1 本ずつ配置する（計 8 本）。

## 点灯ドライブ（1/4 多重化）

8 桁を 2 グループ（A=桁1-4 / B=桁5-8）に分け、4 スロット × 2 グループで多重化する。
毎スロット 2 桁が点灯し、各桁のデューティは 1/4。

![点灯ドライブ アーキテクチャ](../docs/drive-architecture.svg)

### 構成

| ブロック | 部品 | 役割 |
|---|---|---|
| シリアル制御 | **74HCT595 × 2** | XIAO 3 本を 16bit パラレルに展開 |
| カソード（数字選択） | **74141 × 2** | 各グループのカソードバス10本を駆動（0-9） |
| アノード（桁選択） | **高圧ハイサイドSW × 4** | +180V を該当列のアノードへ供給（595 から直接ワンホット） |
| ドット（左右側点） | **MPSA42 × 4** | LA / RA / LB / RB を駆動 |
| アノード抵抗 | 10〜15kΩ × 8 | 各管の電流制限 |

- アノードバス k はグループ A の k 桁目と B の k 桁目を**同時に駆動**（4 本で 8 桁をカバー）。
- カソードは各グループ内で同名どうしをバス接続し、74141 でシンクする数字を選ぶ。
- 桁スロット選択は **595 の 4bit を直接ワンホット出力**で行うため 74139 は不要。

### アノード側 高圧ハイサイドスイッチ（×4）

![アノードHVスイッチ](../docs/anode-hv-switch.svg)

`MPSA92(PNP) + MPSA42(NPN)` の 2 石構成。制御入力 HIGH → Q2 ON → Q1 のベースを引き下げ → Q1 ON → アノードに +180V を供給。
595 の出力（または GPIO）で直接駆動できる。

### 制御ピン（74HCT595 × 2 でシリアル化）

直結なら制御線は 15 本必要だが、**74HCT595 を 2 個デイジーチェーン**して XIAO 側を **3〜4 本**に圧縮する。
595 へのデータ転送はハードウェア SPI（MOSI/SCK）が使える。

**XIAO ESP32-C3 ピン割り当て:**

| 信号 | XIAO ピン | GPIO | 接続 |
|---|---|---|---|
| SER（Data） | D10 / MOSI | GPIO10 | 595 #1 の DS |
| SRCLK（Clock） | D8 / SCK | GPIO8 | 両 595 の SHCP |
| RCLK（Latch） | D7 | GPIO20 | 両 595 の STCP |
| OE（任意） | D6 | GPIO21 | 両 595 の OE（輝度 PWM / 全消灯） |
| RTC SDA | D4 | GPIO6 | I2C RTC |
| RTC SCL | D5 | GPIO7 | I2C RTC |
| ボタン | D3 | GPIO5 | 時刻合わせ等（残り D0-D2 も空き） |

> GPIO8/9 は起動時ストラップピン。SCK=GPIO8 は起動後の出力なら問題なし。
> ラッチには干渉を避けて GPIO20（D7）を使う。

**595 ビット割り当て（16bit = 2 バイト送出）:**

| IC.bit | 割り当て |
|---|---|
| #1 Q0-Q3 | 74141-A BCD（A0..A3） |
| #1 Q4-Q7 | 74141-B BCD（B0..B3） |
| #2 Q0-Q3 | アノードスロット選択 ×4（ワンホット） |
| #2 Q4-Q7 | ドット LA / RA / LB / RB |

- **74HCT595 推奨**（VIH=2V なので 3.3V GPIO で確実に High）。74HC595 を 5V 電源で使うと VIH≈3.5V となり 3.3V GPIO ではマージン不足。
- ブランキングは「アノード 4bit を全 0」または OE で全消灯。**OE に PWM** を入れれば輝度調整になる。
- 残った GPIO は RTC（I2C 2 本）・ボタン・ブザー等に回せる。

### 多重化タイミング

- 全体リフレッシュ 100〜200Hz（スロット 400〜800Hz, 約 1.25〜2.5ms/スロット）
- スロット間に **ブランキング 50〜100µs**（前桁の消イオン待ち＝ゴースト防止、アノード OFF）

### レベルシフト

- XIAO（3.3V）→ 595 は **74HCT595** を 5V 電源で使えば VIH=2V なので直結で確実。
- 595（5V 出力）→ 74141・各トランジスタ base は 5V ロジックで駆動され、最も確実。

### ドット（LHDP / RHDP）

74141 は 0-9 のみのため、ドット（左右の側点）は別系統で駆動する。
ドットは各管内の**別カソード**で、数字と同じアノードを共有する。

![ドット駆動](../docs/dot-driver.svg)

- グループごとに左右ドットをバス接続（**LA / RA / LB / RB の 4 本**）。
- 各バスを高耐圧 NPN（**MPSA42, 300V**）でオープンコレクタ駆動（OFF 時に約 150〜180V を耐える）。
- 直列の `R_dot`（約 47kΩ、要調整）でドット電流を 0.5〜1mA 程度に制限。
- スロットに同期して制御することで、**全 16 ドット（8 管 × 左右）を時分割で個別点灯**できる。

> 数字とドットを同じ管で同時点灯すると、共有するアノード抵抗に両方の電流が流れて
> やや暗くなる。気になる場合はその管のアノード電流を増やす（アノード抵抗を下げる）か、
> 演出上ドットと数字の同時点灯を避ける。

## 設計上の注意点

1. **安全第一** — 基板に 180V が乗る。ブリーダ抵抗で OFF 後に放電すること。
2. **ツェナークランプ** — スイッチノードのスパイクを内蔵スイッチ(40V)以下に抑える。Vz は `V_R(18V) < Vz < 35V` を満たす **27〜30V** を選ぶ（ピーク=5+Vz）。Vz 固定なのでスコープ不要。Dsn は高速回復品(≥60V/1A)。
3. **ダイナミックの再点火** — 桁切替のたびに 170V 以上での再点火が必要。リップルで 170V を割らないよう出力コンデンサとリフレッシュ周波数を設計する。
